Software Engineer

도르팀의 Software Engineer는 게이머가 자신의 플레이를 자동으로 기록하고, 더 쉽게 콘텐츠로 만들고, 자연스럽게 공유할 수 있도록 실제 제품을 만드는 역할을 합니다. Windows 데스크탑 앱부터 영상 처리, 온디바이스 AI/ML 모델까지 기술을 제품으로 구현합니다.

지원하기 커피챗 신청
주요 업무

도르의 Software Engineer는 단순히 서버 API를 만드는 역할이 아닙니다. Windows 데스크톱 앱, 영상 처리, 온디바이스 AI/ML 모델, 제품 품질과 성능 최적화까지 아우르며 기술로부터 실제 유저가 사용하는 제품으로 구현하는 역할입니다.

  • 도르의 Windows 데스크톱 앱 전반을 개발하고 개선합니다.
  • 게임 플레이 자동 녹화, 하이라이트 감지, 클립 생성, 편집 경험을 실제 제품 기능으로 구현합니다.
  • 온디바이스에서 동작하는 비전 AI / ML 모델을 제품 환경에 맞게 최적화하고 튜닝합니다.
  • 정확도뿐 아니라 지연 시간, 메모리 사용량, CPU/GPU 점유율, 안정성까지 고려해 제품을 개선합니다.
  • 다양한 게임 환경과 사용자 PC 환경에서 안정적으로 동작하는 경험을 만듭니다.
  • 데이터와 VOC를 바탕으로 문제를 정의하고, 빠르게 이터레이션하며 제품을 개선합니다.
  • PO, Product Designer, Data Analyst와 함께 문제를 해결하고 실제 유저 지표의 변화를 만들어냅니다.
이 포지션에서 특히 중요한 문제

연구용 모델을 만드는 사람보다 유저가 실제로 쓰는 제품 안에서 모델과 앱을 끝까지 동작시키는 사람이 필요합니다.

  • 게임 이벤트를 정확하게 감지하면서도 실제 플레이에 부담을 주지 않는 문제
  • 다양한 PC 사양과 게임 환경에서 안정적으로 녹화가 동작해야 하는 문제
  • 온디바이스 AI 모델의 정확도와 성능을 동시에 만족시켜야 하는 문제
  • 생성된 영상을 유저가 바로 쓸 수 있을 정도의 품질과 속도로 제공해야 하는 문제
  • 복잡한 기술을 유저 입장에서는 단순하고 자연스러운 경험으로 바꿔야 하는 문제
팀에서 만들어갈 서비스
게임 플레이 자동 녹화 (Record)
  • 유저가 놓치고 싶지 않은 순간을 자동으로 기록할 수 있도록 합니다.
  • 특정 이벤트 기반 자동 녹화 경험을 더 정확하고 자연스럽게 개선합니다.
  • 유저가 "설정하지 않아도 알아서 잘 찍히는 경험"을 만드는 것이 목표입니다.
게임 영상 생성 (Create)
  • 자동으로 생성된 클립을 바탕으로 짧고 볼만한 영상이 만들어지도록 합니다.
  • 유저가 직접 편집하지 않아도 결과물이 만족스럽도록 제품을 설계합니다.
  • "귀찮고 어려운 편집" 없이도 콘텐츠가 완성되는 경험을 만듭니다.
게임 영상 편집기 (Edit)
  • 게임 영상 편집에 특화된 효과와 UX를 설계합니다.
  • 전문 툴처럼 복잡하지 않지만, 결과물의 만족도는 높게 유지하는 제품을 만듭니다.
  • 누구나 빠르게 영상 콘텐츠를 만들 수 있는 제작 경험을 목표로 합니다.
공유 경험 (Share)
  • 만들어진 영상을 친구나 커뮤니티에 쉽게 공유할 수 있도록 합니다.
  • 영상 생성에서 끝나는 것이 아니라, 실제 공유와 시청까지 이어지도록 설계합니다.
우리 팀이 일하는 방식
  • 도르팀은 대표가 원하는 제품이 아니라 고객이 원하는 제품을 만듭니다.
  • 1~2주의 짧은 이터레이션 안에서 문제 정의, 가설 수립, 실행, 측정, 학습을 반복합니다.
  • 데이터와 유저의 목소리를 바탕으로 자율과 책임을 가지고 제품을 만듭니다.
  • 엔지니어는 주어진 명세만 구현하는 역할이 아니라, 문제를 정의하고 결과를 만드는 역할입니다.
  • 빠르게 만들고, 빠르게 검증하고, 빠르게 방향을 조정합니다.
  • 기술적 완성도와 실제 사용자 가치를 동시에 높이는 데 집요하게 몰입합니다.
기술 스택

모든 스택에 대한 경험을 기대하지 않습니다. 다만 빨리 배우고 습득이 가능하기를 기대합니다.

  • Computer Vision 또는 딥러닝 기반 기능을 제품에 적용해본 경험이 있는 분
  • 모델 정확도뿐 아니라 latency, memory, CPU/GPU 사용량까지 최적화해본 경험이 있는 분
